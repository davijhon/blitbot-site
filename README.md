# Sixvort — Blitbot

Sitio web (landing + política de privacidad) de **Sixvort**, empresa de tecnología que
desarrolla soluciones de automatización empresarial sobre WhatsApp Business. Su producto
principal, **Blitbot**, es un asistente de ventas y atención por WhatsApp para PyMEs de
América Latina: responde 24/7, toma pedidos sin errores, organiza clientes (CRM) y
confirma pagos. Software y servicios B2B.

## 🛠️ Desarrollo

Sitio estático, sin build ni dependencias. Para verlo localmente basta con abrir
`index.html` en el navegador o levantar un servidor simple:

```bash
python3 -m http.server 8000
# luego abre http://localhost:8000
```

Tipografías (Space Grotesk + Inter) se cargan desde Google Fonts; el resto (CSS, íconos
SVG y JS) es inline.
